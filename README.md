# Siliguri & Darjeeling Tourism Blog

A comprehensive, modern, and highly responsive blog about tourism in Siliguri, Darjeeling, and surrounding Northeast India destinations.

## Features

### Comprehensive Content
- **Gateway to Northeast**: Complete guide to Siliguri as the gateway to Northeast India
- **Transportation Guide**: Detailed information about reaching via Bagdogra Airport (IXB), NJP/SGUJ railway stations, and NH highways
- **Destination Coverage**: In-depth guides for Darjeeling, Mirik, Kalimpong, Kurseong, Gangtok, and other destinations
- **Three Ts**: Information about Tea, Timber, and Tourism industries
- **Travel Tips**: Essential planning information, permits, budgeting, safety tips
- **Sample Itineraries**: 4-day, 7-day, and 10-day travel plans

### Modern, Responsive Design
- **Mobile-First**: Fully responsive design that works on all devices
- **Modern UI**: Clean, professional interface with smooth animations
- **Accessibility**: WCAG compliant with keyboard navigation and screen reader support
- **Performance**: Optimized for fast loading and smooth scrolling
- **Interactive**: Mobile menu, smooth scroll, fade-in animations, back-to-top button

### Technical Stack
- **HTML5**: Semantic markup with proper SEO meta tags
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **Vanilla JavaScript**: No framework dependencies, lightweight and fast
- **Progressive Enhancement**: Works without JavaScript, enhanced with JS

## Transportation Information

### By Air
- **Bagdogra Airport (IXB)**: 16 km from Siliguri

### By Train
- **New Jalpaiguri (NJP)**: Major railway junction
- **Siliguri Junction (SGUJ)**: City center station

### By Road
- NH 10, NH 55, NH 27, NH 717
- Regular bus services from major cities
- Highway development projects ongoing

## Customization

The blog uses CSS custom properties for easy customization. Edit the `:root` section in `styles.css`:

```css
:root {
    --primary-color: #2c5f2d;
    --secondary-color: #97b339;
    --accent-color: #f4a261;
    /* ... more variables */
}
```
